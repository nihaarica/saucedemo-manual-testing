# saucedemo-manual-testing
Manual test cases for SauceDemo application

## Project Description
This repository contains manual test cases written for the SauceDemo web application (http://saucedemo.com).

## Scope of Testing
- Login Page
- Product Listing Page
- Cart Page
- Checkout Page
- Order Confirmation Page

## Each page contains
Test Case ID
Test Scenario
Test Case Description
Prerequisites
Test Steps
Test Data
Expected Result
Actual Result
Status


## Test Case Details
- Test cases are documented in Excel format
- Includes positive and negative scenarios
- Covers UI, functional, and validation testing

## Tools Used
- Manual Testing
- MS Excel
